Extension to add Bulk Retry feature to Gravity Forms Eloqua

== Description ==

> This plugin requires an <a href="https://eloqua.com/" rel="nofollow">Eloqua</a> account and the <a href="https://briandichiara.com/product/gravityforms-eloqua/" rel="nofollow">Gravity Forms Eloqua Plugin</a>.

= Bulk Retry Failed Submissions =

Retry try your failed submission in bulk! Using the entries screen, select the entries you want to resubmit using the checkboxes next to each of them. Use the Bulk Actions drop down and select Retry Submission to Eloqua. You will be notified how many retries were successful and how many failed again. You can always review the retry log by clicking on each individual entry.

== Installation ==
1. Upload main folder inside the .zip file to your plugins folder, or install using WordPress' built-in Add New Plugin installer (use entire .zip file).
1. Activate the plugin
1. Go to the plugin settings page (under Forms > Settings > Eloqua)
1. Scroll to the Extensions section and find settings for this Extension.
1. Follow on-screen instructions.
1. Click Save Settings.
1. If the settings are correct, it will say so.
